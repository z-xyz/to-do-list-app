# To-Do List App

## Getting Started
  - `pnpm install`
  - `pnpm run dev`
  - website is now running at http://localhost:3000

## Framework & Libraries
  - Next.js (React)
  - Typescript
  - Tailwind CSS
  - ESLint

## Future improvements
  - Editing to-do items
  - Sorting to-do list
  - *`Delete All`* button to delete all to-do items
  - Animation for deleting and adding to-do items
